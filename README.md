### Instalation
```
git clone: https://github.com/M-Hidayatullah/base-admin-laravel-10-skydash.git
```

### composer install
```
composer install
```

### copy .env
```
cp .env.example
```

### generate keys
```
php artisan key:generate
```

### run projects
```
php artisan serve
```
